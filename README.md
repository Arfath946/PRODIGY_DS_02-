Titanic Survival Analysis - EDA Project

Objective

Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset to find patterns and insights related to passenger survival.

Dataset

Titanic Dataset (Kaggle / Provided CSV)

Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn

Steps Performed

1. Loaded the dataset using Pandas
2. Checked missing values and data types
3. Cleaned data (handled null values, removed unnecessary columns)
4. Performed exploratory data analysis
5. Created visualizations to understand survival patterns

Key Insights

- Females had higher survival rates than males
- First-class passengers had better survival chances
- Age and passenger class influenced survival
- Most passengers were between 20–40 years

Conclusion:This project helped in understanding how data cleaning and visualization reveal meaningful patterns in real-world datasets.
